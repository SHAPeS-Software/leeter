# 733+3r

---

...or leeter is a command-line utility to convert those
boring human words into leet hacker ones.

## Features

- leet word conversion

## Building

Run a quick:

```bash
$ go build .
```

now you have `leeter` (or `leeter.exe` on Windows), add it to PATH, or
just run it:

```bash
$ ./leeter

733+3r, a program to convert boring words to hacker ones.
Copyright (c) 2026 SHAPeS
SYNTAX:
	C:\Users\SHAPeS\personal\leeter\leeter.exe [INPUT]
```
